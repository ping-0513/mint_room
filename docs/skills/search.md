# 検索・調べ物のコツ

対象: 検索ツールを使って調べ物をするAI(将来の検索機能・ニュース機能と共通)。

## 手順

1. **検索語は「答えが書いてあるページの言葉」で作る。** 質問文そのままではなく、答えのページに載っていそうな語(製品名・正式名称・エラーメッセージの原文)に変換する。日本語で薄ければ英語でも引く。
2. **鮮度が命の話題は期間を絞る。** 料金・仕様・モデル情報・法制度は1年前の情報が平気で嘘になる。「いつ時点の情報か」を必ず確認し、回答にも時点を添える。
3. **一次情報に遡る。** まとめ記事で当たりをつけ、公式ドキュメント・発表元で裏を取る。まとめ記事同士の一致は裏取りにならない(同じ元ネタの可能性)。
4. **2ソース原則。** 断定して伝えるのは独立した2ソースで確認できたことだけ。1ソースなら「〜とされている(出典1つ)」と弱める。
5. **確度ラベルを使う。** このアプリのニュース機能と同じ語彙で: 公式/報道/噂/推測。噂を事実の顔で伝えない。
6. **見つからないは正当な結果。**「確かな情報は見つからなかった」+「どこまで調べたか」を報告する。埋め合わせに推測を混ぜない。
7. **出典を添える。** マスターが自分で確認できるようリンクを残す。

## 誠実さ

- 検索結果の見出しだけで答えない(本文と食い違うことが多い)
- 自分の学習知識と検索結果が食い違ったら、新しい方を疑いつつ両方を提示して時点を確認する
- 広告・アフィリエイト色の強いページは根拠として格下げする
